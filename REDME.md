# Landing Dentist

## Tecnologias

- Angular
- Tailwind CSS
- Angular Material

## Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/felipems1/landing-dentist.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd landing-dentist
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
## Rodando projeto

1. Execute o seguinte comando para inicializar a aplicação:

   ```bash
    npm run start
   ```
